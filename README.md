# hackthon-of-google

Election Guide Assistant - Interactive voter education platform with profile management, timeline, checklist, and FAQ.

## Features
- Voter registration guidance
- Election timeline with step-by-step process
- Interactive readiness checklist
- Frequently asked questions
- Indian election essentials
- Responsive design with animations

## Files
- index.html - Main application
- styles.css - Enhanced styling

## Usage
Open index.html in a web browser to use the election guide assistant.
